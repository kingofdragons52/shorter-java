# Shorter Java (URL Shortener)

Простой и быстрый сервис для сокращения длинных ссылок, написанный на Java. Проект перенесён с PHP-архитектуры и оптимизирован под высокие нагрузки за счёт отказа от случайной генерации токенов.

## 🚀 Стек технологий
* **Language:** Java 21
* **Database:** SQLite (Драйвер `org.xerial:sqlite-jdbc`)
* **Logging:** SLF4J
* **Frontend:** HTML5, Bootstrap 5 (CDN)

## 🛠 Архитектура и алгоритм (Base62)
В отличие от базовых реализаций, использующих случайный перебор строк и проверку коллизий в цикле `while`, данный сервис использует детерминированный математический подход:
1. Каждая длинная ссылка сохраняется в базу данных SQLite, где ей присваивается уникальный автоинкрементный числовой идентификатор (`id`).
2. Полученный `id` кодируется методом **Base62** (алфавит: `a-z`, `A-Z`, `0-9`).
3. Сгенерированный короткий токен обновляется в БД и отдаётся пользователю.

Этот подход гарантирует полное отсутствие коллизий и стабильное время работы алгоритма ($O(1)$) независимо от объёма накопленных в базе данных.

## 📦 Как запустить проект

1. Клонируйте репозиторий.
2. Откройте проект в IntelliJ IDEA как Maven-проект.
3. Запустите класс `ru.codekitchen.spring.App`.
4. Сервер поднимется локально на порту `8080`: [http://127.0.0.1:8080/](http://127.0.0.1:8080/)
